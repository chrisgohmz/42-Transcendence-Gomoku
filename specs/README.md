# Specs

Use this directory for human-readable Playwright test plans generated by agents or written by hand.

- Start exploratory planning from `tests/e2e/seed.e2e.ts`.
- Name plans `<feature>.plan.md`.
- Use `.e2e.ts` for generated Playwright test files so `bun test` does not pick them up as unit tests.
- Keep each scenario independent so generated tests can run in any order.
